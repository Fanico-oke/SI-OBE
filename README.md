# SI-OBE (Sistem Informasi Outcome-Based Education)

Sistem Informasi untuk manajemen kurikulum berbasis *Outcome-Based Education* (OBE). Proyek ini menggunakan arsitektur monorepo yang memisahkan antara *Frontend* dan *Backend*.

## Struktur Proyek

- `/frontend` - Aplikasi klien berbasis React, TypeScript, Vite, dan Tailwind CSS.
- `/backend` - (Sedang dibangun) API Server berbasis Node.js, Express, Prisma ORM, dan MySQL.

## Prasyarat Lingkungan Pengembangan

Proyek ini direkomendasikan berjalan di atas **Laragon** untuk kemudahan manajemen lokal, dengan persyaratan:
- Node.js (v18+)
- MySQL (v8+)

## Cara Menjalankan

### Frontend
1. Masuk ke direktori frontend: `cd frontend`
2. Install dependensi: `npm install`
3. Jalankan server development: `npm run dev`
